## Overview

This website is my personal portfolio that includes sections about me, my education, my projects, and ways to connect with me. It is built using HTML, CSS, PHP, and JavaScript.

## Features

- Responsive design
- Dynamic project fetching from GitHub
- Smooth scrolling navigation
- Pull-out menu for quick links
- Scroll-to-top buttons

## Setup

To set up the project locally, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/CadenFinley/CadenFinley.com.git
    ```
2. Navigate to the project directory:
    ```sh
    cd CadenFinley.com
    ```
3. Open `index.html` in your preferred web browser.

## Usage

- The header contains navigation buttons to scroll to the "Connect" and "Projects" sections.
- The "Projects" section dynamically fetches and displays repositories from my GitHub account.
- The "Connect" section provides ways to connect with me via social media.
- The footer contains additional links to my social media profiles.
- The "Contact" section includes a form to send me a message directly.

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- [GitHub API](https://docs.github.com/en/rest)

## Authentication

- User registration and login functionality is implemented using PHP and MySQL.
- Passwords are securely hashed using PHP's `password_hash` function.
- Sessions are used to maintain user login state.

## AI Chat Feature

- An AI chat feature is integrated using OpenAI's GPT model.
- Users can interact with the AI after logging in.
- The chat history is maintained during the session for a continuous conversation experience.
